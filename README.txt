# Recipe Book Project

This is a web application that allows users to add, view, and search for recipes.

## Features
- Add new recipes with name, ingredients, steps, and an image.
- View all recipes in a visually appealing format.
- Search recipes by name or ingredients.
- Persistent storage using `localStorage` to store recipes.
- Favicon integration for better branding.
- Logo displayed before and after the main heading image for improved design.

## How to Run
1. Open `index.html` in any web browser.
2. Use the form to add recipes and the search bar to find recipes.
3. The recipes will persist even after refreshing or closing the browser.

## Technologies Used
- HTML
- CSS
- JavaScript

## Additional Features
- Favicon: A logo is used as the favicon to enhance the web app's branding.
- Logo: The logo image is placed before and after the heading image for better visual appeal.

Enjoy using the Recipe Book!
